## ğŸ¯ Quick Start (3 Steps)

### Step 1: Install Dependencies

**Frontend:**
```bash
npm install
```

**Backend:**
```bash
cd backend
npm install
```

### Step 2: Configure Database

1. Create PostgreSQL database: `algovisualizer`
2. Create `backend/.env` file (see `backend/ENV_EXAMPLE.txt`)
3. Fill in your database credentials

### Step 3: Run Application

**Terminal 1 (Backend):**
```bash
cd backend
npm start
```

**Terminal 2 (Frontend):**
```bash
npm run dev
```

**Open Browser:** `http://localhost:5173`
